Tele2 KeepAlive Python script to automatically send a SMS to the ISP, requesting more data.

Tele2 NL has a subscription that allows for unlimited data. Unfortunately they give you a GB budget which they need you to refill manually. This script will send a SMS with a request to refill the data budget on a time interval.

Enter the following command to install the script:

$ bash -c "$(curl -fsSL raw.githubusercontent.com/brutesque/tele2-keepalive/master/install.sh)"
If everything went well, the Raspberry Pi will reboot and SMS messages will appear in the outbox.
